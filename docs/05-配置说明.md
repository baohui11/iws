# 配置说明

配置文件：

- `web/.env.example`
- `file-worker/.env.example`
- `docker/.env.example`

## Web 核心配置

```env
APP_BASE_URL=http://localhost:3000
AUTH_SECRET=
FILE_UPLOAD_TOKEN_SECRET=
DATABASE_URL=
```

- `APP_BASE_URL`：邮件链接、系统外部访问地址。
- `AUTH_SECRET`：会话签名密钥，生产必须强随机。
- `FILE_UPLOAD_TOKEN_SECRET`：文件直传 token 签名密钥，留空时回退 `AUTH_SECRET`。
- `DATABASE_URL`：PostgreSQL 连接串。

## 初始化管理员

```env
SEED_ADMIN_ON_STARTUP=true
SEED_ADMIN_EMAIL=admin@iws.local
SEED_ADMIN_PASSWORD=admin12345
```

Docker 启动时可自动判断是否需要创建初始管理员。

## S3 / MinIO

```env
S3_ENDPOINT=http://localhost:9000
S3_PUBLIC_ENDPOINT=http://localhost:9000
S3_REGION=us-east-1
S3_ACCESS_KEY=
S3_SECRET_KEY=
S3_FORCE_PATH_STYLE=true
S3_PROJECT_FILES_BUCKET=project-files
S3_AVATAR_BUCKET=avatars
```

- `S3_ENDPOINT`：服务端访问地址。
- `S3_PUBLIC_ENDPOINT`：浏览器访问地址，可配置为加密网关地址。
- `S3_PROJECT_FILES_BUCKET`：项目文件桶。
- `S3_AVATAR_BUCKET`：头像桶。

## SMTP

```env
SMTP_HOST=smtp.exmail.qq.com
SMTP_PORT=465
SMTP_SECURE=true
SMTP_USER=
SMTP_PASSWORD=
SMTP_FROM=周报文件系统 <noreply@example.com>
SMTP_TIMEOUT_MS=10000
SMTP_TLS_REJECT_UNAUTHORIZED=true
```

用于忘记密码和用户邀请邮件。

## OA 同步

```env
OA_PGREST_BASE_URL=https://mpdata01.mpgroup.cn:8081
OA_PGREST_TOKEN=
OA_PGREST_PAGE_SIZE=200
OA_PGREST_PROJECT_PAGE_SIZE=500
OA_PGREST_PROJECT_ROLE_PAGE_SIZE=500
OA_SYNC_SCHEDULE_ENABLED=true
OA_SYNC_SCHEDULE_CRON=0 4,13 * * *
```

定时任务运行在 Web 容器内。默认每天 04:00 和 13:00 执行一次全量同步。

## Embedding

```env
DASHSCOPE_API_KEY=
EMBEDDING_API_URL=https://dashscope.aliyuncs.com/api/v1/services/embeddings/text-embedding/text-embedding
EMBEDDING_MODEL=text-embedding-v4
EMBEDDING_DIM=1536
EMBEDDING_BATCH_SIZE=10
EMBEDDING_SERVICE_URL=http://localhost:8010
EMBEDDING_SERVICE_TOKEN=
```

Web 调用 file-worker 的内部 embedding service 生成 query vector。
